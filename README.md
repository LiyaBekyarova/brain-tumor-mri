# Brain Tumor MRI Classifier

A Deep Learning web application designed to classify brain MRI scans into four categories: Glioma, Meningioma, Pituitary, and No Tumor.

Built with Python, TensorFlow 2.18, and Flask 3.1.

## Overview
This project utilizes a fine-tuned VGG16 architecture to provide high-accuracy medical image classification. It features a modern web interface for easy MRI uploads and real-time AI analysis.

### Performance:
* Accuracy: 96%
* AUC Score: 0.97 - 1.00
* Processing Time: < 1 second per image

---

## Tech Stack
* Backend: Flask 3.1.0
* AI Model: TensorFlow 2.18.0, Keras 3.7.0
* Frontend: HTML5, CSS3, Bootstrap 5, Jinja2
* Data Processing: NumPy 2.0.2, Pillow 11.1.0
